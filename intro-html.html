Getting Started with HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content on the web using elements represented by tags.

  • HTML stands for HyperText Markup Language.
  • It is the standard language used to create and structure web pages.
  • HyperText: Allows linking between web pages.
  • Markup Language: Uses tags to define content structure.

2. Structure of an HTML Document

  • HTML documents have a basic structure with the following tags:
  • <!DOCTYPE html> ‚Äì declares HTML5

  • <head></head> ‚Äì metadata, title, links
  • <body></body> ‚Äì main content of the page

  • <strong> / <em> ‚Äì bold / italic text

4. HTML Attributes

  • Attributes provide additional information about elements.
  • Example: <a href="https://www.example.com">Visit Example</a>
  • Common attributes: href, src, alt, title, id, class

5. HTML Comments

  • Comments are ignored by the browser and used for notes in code.
  • Syntax: <!-- This is a comment -->

6. Summary

  • HTML is the backbone of web pages.
  • It defines the structure and layout of content using tags.
  • Learning HTML is the first step in web development.
