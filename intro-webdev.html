Introduction to Internet & Networking

🌍 How Does the Internet Work?

Principle 1: The Need to Share Information

  • Send data from Computer A to Computer B.
  • Direct link via cable → signals as 1s and 0s.

Principle 2: The Problem of Scale

  • 100 computers = too many cables.
  • Solution → Switch (Local Network).

Principle 3: Connecting Networks

  • Use Router to connect multiple networks.
  • Internet = Network of Networks.

Principle 4: Finding Anyone (Addressing)

  • Every device has an IP address.
  • Routers forward data step by step.

Principle 5: Reliability & Fairness (TCP/IP)

  • Data broken into packets.
  • Packets follow rules = TCP/IP protocol.

📌 IP Address

IPv4: 32-bit, example 172.217.16.142

IPv6: 128-bit, example 2001:0db8:85a3::8a2e:0370:7334

📌 MAC Address

  • Unique hardware ID (e.g., 3C:22:FB:A3:B4:C5)
  • Find using ipconfig /all or ifconfig

📌 Port Numbers

  • 0–1023: Well Known (HTTP 80, HTTPS 443)
  • 1024–49151: Registered (MySQL 3306, MongoDB 27017)
  • 49152–65535: Dynamic/Private

📌 DNS (Domain Name System)

  1. Check Cache (Browser → OS → Router)
  2. Ask Resolver (ISP / Google DNS)
  3. Go to Root → TLD → Authoritative Servers
  4. Resolver caches answer
  5. Browser connects to IP
